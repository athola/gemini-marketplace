# Contributing to Gemini Marketplace

We welcome contributions to the Gemini Marketplace! This document outlines the basic steps for contributing to the project.

## How to Contribute

1.  **Fork the repository** and create a branch for your changes.
2.  **Make your changes**, following the existing code style and conventions.
3.  **Run the tests, linter, and formatter** to ensure that your code is correct and consistent.

    ```bash
    make test
    make lint
    make fmt
    ```
4.  **Submit a pull request** with a clear and concise description of your changes.

## Code of Conduct

We have a [Code of Conduct](https://github.com/gemini-rs/marketplace/blob/main/CODE_OF_CONDUCT.md) that we expect all contributors to adhere to. Please read it before contributing.

## Questions?

If you have any questions, please feel free to open an issue on the [Gemini Marketplace issue tracker](https://github.com/gemini-rs/marketplace/issues).
